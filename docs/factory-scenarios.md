# 공장 시나리오

## Factory A

자동차 부품 생산 공장이다. 정상 생산과 낮은 불량률을 기준으로 운영되며, 주로 생산량과 재고 소진 상태를 관찰한다.

## Factory B

배터리 모듈 생산 공장이다. 설비 부하가 높아 진동, 온도, 전류 임계치 초과가 자주 발생한다. 예지보전과 긴급 정비 RPA Task가 핵심 시나리오다.

## Factory C

전장 부품 생산 공장이다. Lot 기반 품질 추적과 출하 보류가 중요하다. 불량률 상승 시 재검사, 재작업, 출하 보류 흐름을 만든다.

## 복합 이벤트

- 원자재 부족 → 생산량 감소 → 납기 지연 → 긴급 보충 RPA
- 설비 이상 → 생산량 감소 → 불량률 증가 → 정비 승인 요청
- Lot 품질 실패 → 출하 보류 → 재작업 지시 → 승인 기반 RPA
