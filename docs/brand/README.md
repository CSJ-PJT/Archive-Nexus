# Archive Nexus Brand

Archive Nexus shares the ArchiveOS geometric mark and uses a cyan-accented Nexus wordmark.

The mark is deliberately flat and two-dimensional. Its `A` contains one cyan
convergence node and one detached lower-right bar. Do not add another white
crossbar through the node, turn the node into a line, or apply gradients,
shadows, bevels, or other 3D treatments.

## Assets

- `archive-nexus-mark.svg` — square mark for app icon, favicon, sidebar, and compact UI surfaces.
- `archive-nexus-lockup.svg` — horizontal README and product lockup.

Use the square mark in compact product surfaces. Use the lockup where the full
name and tagline have enough room. Preserve the dark background and clear space;
do not recolor individual pieces of the mark.

## Palette

| Token | Hex | Usage |
| --- | --- | --- |
| Background | `#071014` | dark manufacturing console background |
| Surface | `#0F1214` | cards and elevated surfaces |
| Text | `#F5F7FA` | mark and primary text |
| Cyan | `#16D6E3` | wordmark, active metrics, graph accents |
| Border | `#1B2126` | panels and dividers |

## Tagline

> Intelligence. Automation. Convergence.

`NEXUS` and the cyan accent distinguish this product from ArchiveOS while the
shared `A` geometry keeps both products in one visual family.
